# Veck Combat Arena 🎮

A fast-paced **FPS web game** built with **Three.js**. Choose your weapon and fight waves of enemy bots!

## 🎯 Features
- **2 Weapons**: Paintball Gun (50 DMG, laser accurate) & Assault Rifle (25 body / 100 headshot)
- **3 Enemy Types**: Standard, Runner (fast), Tank (heavy)
- **Power-ups**: Health packs & Speed boosts drop on the map
- **Movement**: WASD, Space to jump, C to slide, Right-click to aim
- **Dynamic HUD**: Health bar, ammo counter, score, damage pop-ups

## 🕹️ Controls
| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look around |
| Left Click | Shoot |
| Right Click | Aim down sights (AR only) |
| Space | Jump |
| C | Slide |
| R | Reload |

## 🚀 Play Now
Open `index.html` in your browser, or play on GitHub Pages!

## 🛠️ Built With

- Vanilla JavaScript, HTML5, CSS3
